# Changelog

## 0.1.0

Initial release.

- Automatic checking on save with `fault check --format json`
- Inline diagnostics (errors, warnings, info) as VS Code squiggly underlines
- Quick-fix code actions that run `fault fix`
- Status bar showing current issue count
- Watch mode toggle for continuous checking
- Configurable executable path, auto-check on save, and severity filter
- Supported languages: JavaScript, TypeScript, Python, Go, Java, Rust
